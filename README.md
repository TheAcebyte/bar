# bar

## About
*bar* is a **simple**, but **structured** programming language made with *TypeScript*, it is highly inspired by *Pascal*, hence the name.

> ### Example
> ```
> PROGRAM HelloWorld
> BEGIN
>    PRINT "Hello, World!"
> END
> ```
> For more examples, you can look at the `examples/` directory.
